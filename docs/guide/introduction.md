# 介绍

一个用于逃离鸭科夫的自定义模型模组。

![鸭科夫文档封面](/images/鸭科夫文档封面.png)


::: tip 提示
目前最新版的 Duckov Custom Model 版本号为<VersionCard />
:::


## 项目概述

**自定义模型管理器** (Duckov Custom Model) 是一个强大的模组，允许玩家为游戏添加和管理自定义模型。无论你是想为角色、宠物或 AI 角色使用自定义模型，这个模组都能提供完整的解决方案。

## 主要特点

- 🎨 **自定义模型替换**：轻松替换游戏中的角色模型
- 🎮 **图形化管理界面**：直观的模型选择界面
- 🔍 **模型搜索**：快速查找想要的模型
- ⚡ **快速切换**：游戏中实时切换模型，无需回到主菜单切换MOD
- 🚀 **高效更新**：增量更新机制，只刷新变更内容
- 🎯 **多目标支持**：支持角色、宠物和 AI 角色模型

## 系统要求

- Windows / macOS 
- 逃离鸭科夫游戏本体 (Escape from Duckov)
- HarmoryLib 框架支持
